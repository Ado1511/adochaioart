
# Project Blueprint

## Overview

This project is a personal portfolio website for an Art Director. The design is clean, modern, and creative, with a focus on visual storytelling. The color palette is inspired by pastels, and the overall aesthetic is light and airy.

## Design and Features

### General
- **Color Palette**: Soft pastels (e.g., lavender, mint, peach) contrasted with a clean white or light gray background.
- **Typography**: A combination of a stylish serif font for headings (Playfair Display) and a clean sans-serif for body text (Lato).
- **Layout**: A responsive, single-page layout with clear sections for "Selected Works," "About," and "Contact."
- **Interactivity**: Smooth scrolling for navigation links and subtle hover effects.

### Header
- A fixed header containing the logo and navigation links.
- The logo is the Art Director's name in a stylized font.
- Navigation links smoothly scroll to the corresponding sections.

### Hero Section
- A prominent headline and a brief, engaging tagline.
- A pastel-colored background.

### Selected Works
- A grid of project cards.
- Project cards are full-width and include a header, description, and image gallery.
- Image galleries can have two, three, or four columns, depending on the number of images.
- Hovering over a card reveals a subtle animation.
- An image modal that displays a larger view of gallery images when clicked.
- A "View All Projects" button that links to a separate `works.html` page.

### About Section
- A two-column layout featuring a profile picture and a detailed description of the creative process and strengths.
- The profile picture is circular and has a colored border.
- A styled list of core strengths.

### Contact Section
- A redesigned, modern section with a distinct background color.
- A clear call-to-action to get in touch.
- A prominent email link styled as a button.
- Social media links (WhatsApp, LinkedIn, Behance, Instagram) to encourage further connection.
- A separate copyright footer.
